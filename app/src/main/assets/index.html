This application is developed to provide OCR (Optical Character Recognition) text scanning services. The application works both offline and online depending on feature usage.

2. Information We Do NOT Collect

This application does not intentionally collect personal user information such as:

  • Full name

We do not store scanned images permanently on any server. All OCR processing is performed locally on user device unless third party services are used.

3. Information That May Be Processed

The application may process images selected by the user for text recognition purpose. These images are used only for OCR processing and are not uploaded to our servers.
